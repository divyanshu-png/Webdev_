This is a paragrapdh about External Styling in HTML documents.
To use an external stylesheet in HTML, you need two things: a separate CSS file and a specific HTML tag to link them together. This method is considered a best practice for clean, maintainable code.

Key attributes:

  • rel="stylesheet": This is a required attribute that tells the browser the relationship between the current HTML document and the linked file is a stylesheet
  • href="styles.css": This attribute specifies the path or location of your CSS file. Adjust the path as necessary if your file is in a different directory

Thus we can apply both internal/ inline and external styles to same elements at the same time and it will work on priortiy basis on which one to apply if they clash
A Css property defined with '!important' overrides any other declaration.
Inline Styles(css inside the element as an attribute) comes after that.
And after that comes the Internal (css written in the head element of the html as an style element) and then the external stylesheet(stylesheet added as a link in the head as well using a link element and href, rel attribute ) in priority order.
