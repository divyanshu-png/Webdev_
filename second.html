Relative and Absolute paths of elements in the hard disk


We can add either relative or absolute path to access elements stored locally on the system.
The absolute path looks like this
D:/New Folder 3/WEBDEVELOPMENTBOOTCAMP/introduction.html

The relative path is used to link to files within the same website/project, based on the current document's location.
The relative path looks like either of the following:
  • ./image.jpg :Refers to a file in the current directory(Current directory bole toh vo folder jisme apna file maane ke html file hai)
  • ../style.css: Refers to a file in the parent directory (one level up from the current directory).
  • /about.html: Refers to a file at the root directory of the website (this is a root-relative path)

IFrames

Iframes are used to embed another HTML document rather than just linking it. It incldes a 'src' attribute, value set as the link to the other document. Width and heigth attribute to set dimensions of the Iframe in pixel or percentages. Title can also be used for screen readers and SEO purpose.
Embedding content from untrusted sources can pose security vulnerabilities, which can be mitigated with attributes like sandbox. It requires a value—either an empty string (sandbox="") to apply all restrictions, or a space-separated list of pre-defined values (allow-scripts, allow-forms, etc.) to selectively lift them
