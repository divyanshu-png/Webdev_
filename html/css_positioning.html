There are a total 4 ways of poitioning in CSS:

  • Static Positioning:

    The by default positioning of the element that doesn't need to be explicitly mentioned.


  • Relative Positioning:

    The most used kind of positioning. The element is positioned relative to its normal position in the document flow. Offsets move the element without affecting the space it originally occupied; other elements will not "fill the gap". Frequently used as a container for absolutely positioned child elements.
    Relative means it is relative to it's supposed location(that shouldve been the case in static )




  • Absolute Positioning:

    Position relative to the nearest positioined ancestor or top left corner of the webpage. Absolute position element ko normal document flow se bahar nikal deta hai. Yeh element apne sabse kareebi 'positioned' (relative/absolute) parent ke hisaab se top, bottom, left, right properties use karke kahin bhi rakha ja sakta hai.

  • Fixed Positioning

    Fixed positioning in CSS is a method used to anchor an element to a specific location relative to the browser's viewport. Unlike other positioning types, a fixed element stays in the same place even when the page is scrolled.
