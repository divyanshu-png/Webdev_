4. Chaining selectors
Apply where all selectors are true. 
        selector1selector2...{ #without space
            color: whatever
        }
This is a simple yet effective tool to select a very specific element like:

h1#title.heading-big

If youre chaining any other selector with an element selector, the element selector needs to be the first one to be mentioned otherwise itll be difficult to understand.
