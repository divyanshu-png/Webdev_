Flexbox container properties

The properties that can be applied to the flex container or the container that is set to flex.

1. display
This property initializes the layout context. It defines whether the element behaves as a block or inline flex container.
flex: Activates a block-level flex container
inline-flex: Activates an inline-level flex container
2. flex-direction
This property establishes the main axis, dictating the direction in which flex items are laid out.
row (default): Left-to-right in standard English writing mode.
row-reverse: Right-to-left orientation.
column: Top-to-bottom layout.column-reverse: Bottom-to-top layout.
3. flex-wrap
By default, flex items will force themselves onto a single line. This property controls whether items wrap into multiple lines.
nowrap (default): All flex items stay on one single line. Not very useful as all the elements might not be visible to the user.
wrap: Container Items break onto multiple lines from top to bottom if space is insufficient. Important and useful.
wrap-reverse: Items break onto multiple lines from bottom to top.
4. justify-content
This property handles alignment along the main axis,(that is set to the x axis when the rox flex direction is taken and the y-axis when the column flex direction is take) distributing any remaining free space. It can have the following properties:
flex-start (default): Items pack tightly toward the start edge of the container.
flex-end: Items pack tightly toward the end edge.
center: Items center perfectly along the line
space-between: Items display with equal spacing; first and last items hug the outer edges and use up all the available space of the container.
space-around: Items display with equal space around them; outer margins are half the size of inner gaps.
space-evenly: Items display with identical spacing between them and the outer edges.
