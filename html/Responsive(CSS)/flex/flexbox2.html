Flex-GROW AND Flex-SHRINK

flex-grow and flex-shrink dictate how flexible items adjust their sizes when the container is larger or smaller than the combined size of the items. flex-grow handles extra empty space by expanding items, while flex-shrink handles a lack of space by shrinking items to prevent overflow even beyond the flex basis mentioned.

We can write the flex-grow, shrink and basis value separated by space in the property "flex" itself as
flex: 1 1 0;
where the flex basis 0 mean that every value is set to equal flex-basis(width as in the row flex direction). We can futher shorten this and write the flex as simply 1 to make the same implementation.
